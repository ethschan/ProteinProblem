## Overview

This repository is to document "Protien Problem", a Grade 9 personal project to develop a handheld urinalysis device for kidney patients that suffer from chronic kidney diseases, motivated by personal experience as a patient.

More information about the project can be found [here](https://csfjournal.com/volume-2-issue-5/2020/3/11/protein-problem-a-digitized-at-home-urinalysis-device-to-aid-patients-suffering-from-kidney-disease).
